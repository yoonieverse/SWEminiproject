This mini-project automates the process of verifying whether famous books exist on Wikipedia. It reads a bunch of book tiles from an input Excel sheet, queries Wikipedia for each title, and then writes the results to a second output spreadsheet making each book to be Verified (found on the Wikipedia site) or NOT FOUND (not found on the wiki site).


Project Files:
books_input.xlsx -- Input spreadsheet(list of book titles, authors, and genres to verify)
books_verified.xlsx -- Output spreadsheet(generated by the script with verification results)
wikipedia_book_checker.py --  Main Python script (reads input, queries wikipedia, writes output)

Note: Download books_input.xlsx & update values accordingly


Install:
pip install openpyxl requests   //  python3 -m pip install openpyxl requests
python3 -m pip install python-dotenv


Running the script:
python wikipedia_book_checker.py    //      python3 wikipedia_book_checker.py
