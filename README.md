This mini-project automates the process of verifying whether books exist on Wikipedia. It reads book tiles, author names, and genres from an input Excel file, then queries the Wikipedia API for each title to determine whether a matching title exists. The results are exported to a second excel file, where each book is labeled as VERIFIED (found on the Wikipedia site) or NOT FOUND. For books written in the non-english language, a GEMINI AI API is used to transalte/normalize the titles into English before querying Wikipedia.


Project Files:
books_input.xlsx -- Input spreadsheet(list of book titles, authors, and genres to verify)
books_verified.xlsx -- Output spreadsheet(generated by the script with verification results)
wikipedia_book_checker.py --  Main Python script (reads input, queries wikipedia, writes output)

Note: Download books_input.xlsx & edit values 


Install (python/python3):
python3 -m pip install openpyxl requests
python3 -m pip install python-dotenv

Running the script:
python3 wikipedia_book_checker.py
