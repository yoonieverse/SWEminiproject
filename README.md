This mini-project automates the process of verifying whether books exist on Wikipedia. It reads book titles, author names, and genres from an input Excel file, then queries the Wikipedia API for each title to determine whether a matching title exists. The results are exported to a second excel file, where each book is labeled as VERIFIED (found on the Wikipedia site) or NOT FOUND. 


# Project Files:
books_input.xlsx -- Input spreadsheet(list of book titles, authors, and genres to verify)
books_verified.xlsx -- Output spreadsheet(generated by the script with verification results)
wikipedia_book_checker.py --  Main Python script (reads input, queries wikipedia, writes output)

Text files (Excluded from core implementation):
Note.txt -- Additional information utilized for testing and validation; This file is not required for execution or evaluation.
